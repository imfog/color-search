About Me and Color Search

Hi, I'm Imtiaz. I built Color Search, the search engine for colors and images to get into the world of search engines. This is a very basic implementation of how search engines work but yet it manages to do something very cool.

Why did I build it? Primarily to learn. I tried finding something similar. Couldn't find anything I liked. Moreover, I've been wanting to build something cool. So, I went ahead and built one image/color search engine myself.

The algorithm used to search has been heavily influenced by this blogpost.

The frontend has mostly been built using the code snippets from here.
